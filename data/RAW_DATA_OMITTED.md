# Raw data omitted from Git

Raw NIH ChestX-ray14 and VinDr-CXR/VinBigData-derived images are not redistributed in this repository. Users should obtain the original public datasets from their official sources and follow the original license and citation requirements.

This repository contains code, configuration, metadata/split files, prediction CSVs, result tables, final manuscript materials, and figure source data generated from the completed run.
